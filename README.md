## ğŸ“Œ Project Overview

Finding movies similar to your taste can be difficult when there are thousands of options available. This project solves that problem by recommending movies based on **content similarity**.

The system analyzes the following movie attributes:

* Overview (story description)
* Genres
* Keywords
* Cast

Based on these features, it recommends the **top 5 most similar movies** to the one selected by the user.

---

## ğŸ§  Machine Learning Approach

* **Recommendation Type**: Content-Based Filtering
* **Text Processing**: NLP (tokenization, text cleaning)
* **Vectorization**: CountVectorizer
* **Similarity Measure**: Cosine Similarity

This approach does not require user ratings and works purely on movie content.

## ğŸ§  How It Works (Step-by-Step)

1. Load and merge movie and credits datasets
2. Clean and preprocess text data
3. Combine key features into a single `tags` column
4. Convert text data into vectors using CountVectorizer
5. Compute cosine similarity between movies
6. Recommend top 5 similar movies
7. Display results with posters and metadata using Streamlit
